자기소개

졸업 과제

안드로이드 예비 개발자 심우태입니다.
졸업 과제로 경주 연합 생활관 애플리케이션을 개발한 경험이 있습니다.
졸업 과제를 통해 안드로이드 애플리케이션 개발에 대한 기초를 다지고 실제로 애플리케이션을 만들어 볼 수 있는 경험을 쌓았습니다.
경주 연합 생활관 애플리케이션은 학생들이 생활관 내에서 필요한 정보를 쉽게 접할 수 있도록 도와주는 애플리케이션입니다.
해당 애플리케이션에선 drawer navigaion 기능으로 메뉴 기능을 구현하였으며, 공공데이터 API를 이용하여, 버스 도착 정보를 표시하는 기능을 구현하였습니다.
네이버 지도 API를 이용하여 지도 화면을 구현하였고, 버튼을 클릭 시, 해당 위치의 정류장 위치와 내 위치를 파악하는 기능을 추가하였습니다.
안드로이드 스튜디오를 주로 사용하며 Java를 주로 사용하였기에 Java에 익숙합니다.

협업 경험

졸업 과제

주피터 노트북과 파이썬을 통한 데이터 분석 조별 과제와 Unity를 이용한 게임 개발 프로젝트 조별 과제를 통한 협업을 하며 팀원들과의 협업에 익숙합니다.
UNITY VERSION CONTROL이나 구글 슬라이드와 같은 협업 도구를 이용하여 작업한 경험이 있습니다.
